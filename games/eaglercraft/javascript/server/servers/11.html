█ B2 █ THE BEST Half-Anarchy Eaglercraft Server █ PVP switch, Trading system, and Chest locking! █


IP: wss://mc.b2server.tk

Description:
B2 - A Half-Anarchy Eaglercraft Server. Check out the website: https://b2.cloudns.nz/ Server limit of 32 players. Running on a Raspberry Pi 4 4GB, so it can get a little laggy. You can build starting 32 blocks away from spawn. To prevent stealing, place a sign on your chests, hoppers, anvils etc. Right click on the sign to add other players. Crouch+Right Click on a player to start a secure item trade with them. MAKE SURE you have an empty inventory slot selected. (it's a major bug that will be fixed in the future). Commands: /spawn (return to server spawn) /msg (private messaging) /pvp (enable pvp for just you) /who (list online people) /time (in game time) /afk (mark yourself as afk) /where (get your coordinates) /compass (get your facing direction) /call (ask to be teleported to player) /lagg tps (show server tickspeed) /sethome (set your home) /home (teleport to your home) /return (go to the place you died or teleported from) /pd (teleport to a random location) /warps list (list warps) /warp (to use warps) Have Fun :)
